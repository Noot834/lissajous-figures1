# lissajous-figures
программа на Python для симуляции фигур Лиссажу в реальном времени с возможностью изменять параметры
 

Установка
1. Клонировать репозиторий
2. Создать виртуальное окружение: "python -m venv venv"
3. Активировать: "venv\Scripts\activate" (Windows) или "source venv/bin/activate" (Linux/macOS)
4. Установить зависимости: "pip install -r requirements.txt"
5. Запустить: "START.py"
